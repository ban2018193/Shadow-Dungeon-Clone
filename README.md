Shadow Dungeon (Bagel Java Game Clone)
A clone of Shadow Dungeon, built in Java using the Bagel game framework.
The project demonstrates object-oriented programming (OOP) principles including inheritance, polymorphism, and encapsulation.

It features game entities such as players, enemies, weapons, and interactive objects, each modeled as independent classes.

Game logic includes movement handling, collision detection, event management, and scene transitions.

Key concepts practiced:
Object-oriented design (classes, interfaces, inheritance)
Separation of concerns and modular architecture
Sprite rendering, input handling, and game loop control with Bagel
Code organization across multiple source files and packages
